个人博客Fan主题美化

       前言

        本篇文章主要是记录下在美化Fan中的过程还有我自己在搭建好之后美化过程中出现的各种问题作以记录。

一、logo图标

        搭建好博客之后，可以更改自己设计好的logo图标，也可以去网上找相应的图片格式的话可以是PNG、IOC。可以在easyicon、iconfont去找。找到图标之后接下来是设置，设置的话根据主题的不同设置方式也有点不同。如Next主题找到博客的本地目录下的/themes/Next/_config.yml中更改small和medium两个地方的路径即可。

favicon:
  small: /images/16x16.png
  medium: /images/32x32.png
  apple_touch_icon: /images/128x128.png

Fan主题logo设置找到博客的本地目录下的/themes/Fan/_config.yml中更改favicon节点路径，avatar节点是设置博客中的照片
# Miscelaneous
favicon: /lol.png

# avatar
avatar: /avatar.png

二、主题美化

        可以去Hexo主题官网中下载自己喜欢的主题，我安装的主题是Fan星空主题。

1.安装
git clone https://github.com/fan-lv/Fan.git themes/Fan

安装完之后修改位于博客根目录下的_config.yml内的theme选项值为Fan

2.更新
cd themes/Fan
git pull
3.需要的依赖包
npm install hexo-renderer-jade
npm install hexo-renderer-stylus
4.问题

       如果报错 appId undefined,需要把 themes-Fan->_config.yml 中 algolia_search:enable 改成 false

5.说明

       支持文章置顶功能，在需要置顶的文章加入 top: 1，数值越大越靠前。
