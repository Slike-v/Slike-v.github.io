SpringBoot访问JSP

       前言

        Spring Boot默认是不支持jsp页面的访问的。这一点有点不同，SpringBoot如果需要访问JSP、html等页面需要导入相关的依赖。

快速搭建SpringBoot框架

Spring initializr 是Spring 官方提供的一个用来初始化一个Spring boot 项目的工具。

使用集成工具idea,创建Spring Initializr，选择Spring Initializr，选择JDK1.8,Next。

设置好jdk版本和Spring initializr项目,Next

现在建立的是web项目,选择Spring Web Starter和相应的SQL，Next

选择相应的依赖关系。我们搭建的是Web项目使用的是mysql数据库。

下一步

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>
    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>2.1.6.RELEASE</version>
        <relativePath/> <!-- lookup parent from repository -->
    </parent>
    <groupId>com</groupId>
    <artifactId>springboottest</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <name>springboottest</name>
    <description>Demo project for Spring Boot</description>

<properties>
    <java.version>1.8</java.version>
</properties>

<dependencies>
        <!--Web的依赖-->
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-web</artifactId>
    </dependency>
    <!--JSP依赖 JSP必须的依赖-->
    <dependency>
        <groupId>org.apache.tomcat.embed</groupId>
        <artifactId>tomcat-embed-jasper</artifactId>
    </dependency>
     <!-- servlet依赖 -->
    <dependency>
          <groupId>javax.servlet</groupId>
          <artifactId>jstl</artifactId> 
    </dependency>
    <dependency>
            <groupId>javax.servlet</groupId>
         <artifactId>javax.servlet-api</artifactId> 
    </dependency>
    <!--mysql依赖-->
    <dependency>
        <groupId>mysql</groupId>
        <artifactId>mysql-connector-java</artifactId>
        <scope>runtime</scope>
    </dependency>
    <dependency>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-test</artifactId>
        <scope>test</scope>
    </dependency>
</dependencies>

<build>
    <plugins>
        <plugin>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-maven-plugin</artifactId>
        </plugin>
    </plugins>
</build>
</project>

SpringBoot的相应依赖已经全部搭建好了,不过SpringBoot要访问JSP页面需要在导入相应的依赖使用了Spring Initializr创建的SpringBoot之后。控制器和application.properties都已经创建好了。也可以使用yml文件来书写配置。
